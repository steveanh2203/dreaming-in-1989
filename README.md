# Dreaming in 1989

Retro American 80s/90s nostalgia website demos.

## Projects

- `mockup 1` - supermarket circular / nostalgia market demo.
- `mockup 2` - alternate website direction.
- `awesome-design-md` - cloned design reference material used during ideation.

## Run Mockup 1

```bash
cd "mockup 1"
npm install
npm run dev
```

## Run Mockup 2

```bash
cd "mockup 2"
npm install
npm run dev
```
