# Project Working Rules

## Merch Artwork Defaults

- For product artwork that will be applied to physical goods such as mugs, pillows, apparel, tote bags, wall decor, stickers, or similar print-on-demand items, default to a transparent-background print file.
- Avoid full rectangular colored backgrounds unless the user explicitly approves a poster/sticker/label look.
- For the Dreaming in 1989 product-concept workflow, generate concepts on a flat green chroma-key background only to make background removal reliable. The green background is temporary and must never be printed on the product.
- Product concepts do not require the storefront logo. Prioritize an original, product-appropriate 1980s-1990s American concept unless the user explicitly requests branding.
- The product blank should usually provide the base color. For example, a white mug should use the mug surface as the white background instead of printing a cream or tan rectangle behind the design.
- Preserve intentional fills inside the artwork, such as lettering, badges, checker patterns, object highlights, and vintage texture. Remove only the broad background field that would make the design look pasted onto the product.
- Final print-ready artwork should be saved as PNG with alpha when transparency is needed. Keep a white-background preview and a checkerboard preview when useful for review.
- Before considering a merch concept approved, generate or request an actual product mockup using the transparent artwork so the user can judge how it sits on the physical product.

## Product Image Workflow

- Brainstorm a product-appropriate concept and generate it on a flat green chroma-key background.
- Remove the green background and validate a transparent PNG before applying the artwork to Printful.
- Apply only the transparent artwork to the selected Printful product, then download every useful mockup angle Printful provides.
- Remove the background surrounding each downloaded Printful product mockup and compose the isolated product inside a centered `1:1` frame.
- Generate lifestyle images with the background-removed concept PNG as the required primary artwork reference. Use the Printful mockup only as a secondary reference for the physical product, proportions, and print placement.
- Lifestyle images must be `1:1`, preserve the exact product artwork and print placement, and use believable American 1980s-1990s environments.
- The PDP hero lifestyle/story frame should use a product-related lifestyle image that fits the real frame tightly without distortion, bad cropping, or losing the main product. Rewrite the right-side copy for that exact product instead of reusing generic text.
- Apparel lifestyle images should use period-appropriate American models in natural daily-life scenes; a front-facing pose is not required.
- Target 10 storefront images for a complete standard product package: 4 Printful/product views, 1 product detail, 1 PDP hero lifestyle/story image, and 4 square lifestyle/lookbook images. Download additional useful Printful views when available. The transparent concept artwork is a production asset and is not counted among the 10 storefront images.

## Printful Workflow Guardrails

- Do not publish products, create real orders, confirm orders, change pricing in bulk, or replace live product print files without explicit user approval.
- Mockup generation, catalog checks, availability checks, and local report updates are allowed as review/prep work.
