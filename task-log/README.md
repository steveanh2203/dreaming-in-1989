# Task Log

Kanban notes for daily project work.

## How to Use

- Put today's active work in `Today`.
- Move one item at a time into `In Progress`.
- Use `Review / QA` for work that needs checking before it is considered done.
- Use `Blocked` only when progress needs a decision, missing asset, or external setup.
- At the end of the day, move finished items to `Done` and copy unfinished items into the next daily board.
